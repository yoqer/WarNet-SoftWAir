# WarNet-SoftWAir
El: SoftWAir, es un sistema de SoftWare con IA, modular y Autoescalable por su tecnología independiente, basado en la amalgama y variabilidad de vuelos y tecnología terrestre, para la comunicación con tecnologías Satelitales, Wifi, mmWave, 4G, 5G, 6G, 7G, GPS, GPRS, Bluetooth, LoraWan, etc... 
Con orientacion Post-Cuantica y Cualidades Autónomas y Autoreplicantes.
