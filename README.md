# WarNet-SoftWAir
El: SoftWAir, es un sistema de SoftWare con IA, modular y Autoescalable por todas sus tecnologías independientes, basado en la amalgama y variabilidad de vuelos y las tecnologías terrestres, para la comunicación con tecnologías Satelitales, Wifi, mmWave, 4G, 5G, 6G, 7G, etc; GPS, GPRS, Bluetooth, Lora, LoraWan, etc... 
          -Con orientacion Post-Cuantica y Cualidades Autónomas y Autoreplicantes Escalantes.
